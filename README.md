QML.Chart
=========

Chart with QML(Qt5.x)
Hello!

A Project used for QML Chart.

you can also use Qt Offical Chart:
  QtCharts:http://doc.qt.digia.com/QtCharts/

This chart use QML.Canvas(a Canvas like Html5) and Javascript.

Hello！
这是一个用于QML的chart控件。
